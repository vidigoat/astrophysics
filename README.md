# Causal Structure in Galaxies: Comparing Simulations and Observations

Code for the paper *"Causal Structure in Galaxies: Comparing Simulations and Observations"* by Vidit Patankar. The work uses causal discovery (the FCIT algorithm) to learn the directed relationships between galaxy properties — mass, luminosity, size, metallicity, gas content, and so on — and compares what we find in real observations (NSA optical galaxies and ALFALFA HI-selected galaxies) against what we find in cosmological simulations (TNG50). The goal is to see whether simulations reproduce the same causal structure that the real universe shows.
